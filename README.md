Proyecto del Espacio para el desarrollo laboral y tecnologico.

+ **Python 3.6.5**
+ **Pip3 18.0** 
+ **Xampp 3.2.2**
+ **MySQL Connector/Python==8.0**
+ **Django 2.1.1**
+ **Filter**
+ **Django Rest Framework**
+ **Gunicorn**
+ **White Noise**
+ **PyMySQL**
+ **My SQL client**
+ **Decouple**
+ **Virtual env**
