# ProyectoEspacio
Proyecto del Espacio para el desarrollo laboral y tecnologico.

## Dependencias

+ **Python 3.6.5**
+ **Pip3 18.0** 
+ **Xampp 3.2.2** 
+ **MySQL Connector/Python==8.0**
+ **Django 2.1.1**
+ **Filter**
+ **Import-export 1.1.0**

> sudo pip install
> sudo pip install django==2.1.1
> sudo pip install django-filter
> sudo pip install django-import-export
